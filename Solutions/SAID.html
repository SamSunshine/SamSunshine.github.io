Satellite Anomaly Injection & Detection Testbed

Integrated Software Components

Developed software components to integrate with and utilize existing industry open source software to inject and detect anomalies in satellite simulations.

Problem statement




Satellites perform various vital functions, including communications, weather prediction, geolocation, defense, etc. Accurate data must flow freely between the ground and the satellite in these complicated systems via uplinks and downlinks. When strange behaviors or anomalies occur, the errors must be identified and corrected before a disaster occurs. Sometimes these anomalies result from errors in the hardware or software, issues introduced by the environment, or an attack by a hacker.

The solution




For a solution, our team utilized machine learning and simulation tools to inject and detect these abnormalities by comparing actual observed values with the predicted intervals of telemetry data. Our simulation tools allow Aerospace engineers to configure and deploy anomalies in real-time environments. We utilized machine learning to detect these anomalies by comparing actual observed values with the predicted intervals of telemetry data.
